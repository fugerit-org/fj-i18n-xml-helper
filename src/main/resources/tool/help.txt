# XML label extract helper
  
Utility to extract xml attributes or elements label and converting to property keys.
  
Example : java -jar target/dist-fj-i18n-xml-helper-*.jar \
	--input-xml src/test/resources/config/test/xml \
	--output-xml target/test-output \
	--output-properties target/test-properties.xml \
	--convert-config src/test/resources/config/test/convert-config.xml
	  
## parameter help  

	--input-xml <arg> The input path to XML, if a directory all files will be handled recursively. [default:none]

	--output-xml <arg> The output path where to produce the XML, if input is a directory, output MUST be too. [default:none]

	--output-properties <arg> The output xml properties containing the processed labels. [default:none]

	--convert-config <arg> If set to `true`, the 'info' columns of tool parameters will be ignored.

	--filter-ext <arg> A extension filter, if inputXml is directory, only files matching the extension will be processed.

	--catalog-rule-id <arg> if set, the rule catalog in convert-config will be used instead of the default one.

	--help <arg> Print help about the tool. [default:none]
